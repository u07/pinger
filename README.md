# Скрипт для мониторинга сети: порты, URL, системное время и пинг со звуковым оповещением

Скрипт с GUI на HTA для отслеживания доступности в сети устройств и сервисов, со звуковым оповещением.

Работает без установки, без зависимостей, без Zabbix. Умеет проверять доступность хостов по ICMP, состояние TCP-портов (доступность баз данных, веб-серверов, почты), HTTP-статус сайтов и сервисов, расхождение системного времени. 

Когда что-то идёт не так - пищит через колонки. 

Список адресов для мониторинга задаётся внутри, в самом начале. Кодировка utf-8.
